Knoxx Security Standard

Knoxx is a security standard for maximum security in Linux, from using a specially compiled kernel to hardened replacements for common Linux applications. Knoxx covers most areas of attack and is an active standard used by William to secure Linux systems.

Quinn AI

Quinn is a powerful bot designed to detect sexually explicit behavior and prevent it from happening around children. It is owned by the Solus Project with most of the contributions made by William Kenzie.

Secure Communication Protocol

Secure Communication Protocol, aka Rusty, is a highly encrypted format for storing files and text. It uses a combination of PGP, AES-512, and several layers of encryption based on intimate details of the recipient's machine.
